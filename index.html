Jquery ScrollTop Button Example

This is an example of the Jquery ScrollTop Button plugin using the default options of a 20px top offset and a html entity ^ as the inner button symbol.

Scroll the bottom lipsum text. You should see the ScrollTop button appear on the bottom right corner.
